# Exercício 01 do Bootcamp GoStack 11 da Rocketseat

O exercício proposto está relacionado à criação de uma API para manutenção de projetos. A API em questão será desenvolvida utilizando Node.js

## Instalação e execução

1. Faça um clone desse repositório;
2. Entre na pasta rodando `cd bootcamp-gostack-exercicio-01`;
3. Rode `yarn` para instalar as dependências;
4. Rode `yarn dev`.
